This directory contains portions of the Thin Plate Spline editor

source code is available at http://elonen.iki.fi/code/tpsdemo/

This version has been edited as part of the MCC-LiDAR project

MCC-LiDAR projects SVN.

The source code has subsequently been edited by Pete Bunting
to be part of spdlib where the namespace has been updated to
spdlib::tps and bug fixes applied to remove c++ warning.
